# Ejecutar en desarrollo:

1. Clonar repositorio

```bash
git clone https://github.com/AnthoFu/03-pokedex.git 
```

2. Ejecutar

```bash
npm install
```

3. Tener Nest CLI instalado

```
npm i -g @nestjs/cli
```

4. Levantar la base de datos

```bash
docker-compose up -d
```

5. Conectarse a la base de datos de mongoDB (En mi caso utilic√© TablePlus)

mongodb://localhost:27017/nest-pokemon
